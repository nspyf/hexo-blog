安装Manjaro

  1. 准备一个容量足够的空U盘（用于制作USB启动盘）
  2. 做好备份（不要用上文提到的U盘）
  3. 清华大学开源软件镜像站下载Manjaro镜像
  4. 下载Rufus，制作USB启动盘。插入U盘，打开Rufus，选择镜像，开始（注意，这一步会清除U盘数据）
  5. 分配磁盘空间：此电脑->管理->磁盘管理，选择空间充足的盘进行分区（右击选择压缩卷）
  6. 关闭Windows快速启动
  7. 重启，按F2（或其它）进入BIOS，关闭安全启动secure boot
  8. F12（或其它）进入引导菜单，选择USB启动
  9. 按提示进行配置安装。配置分区时，取代一个分区，选择之前空闲的分区。如果联网了，需要更换国内源，不然下载很慢，具体方法如下，按F12打开下拉终端，输入sudo pacman-mirrors -i -c China -m rank，勾选几个，再执行sudo pacman -Syu更新，即可。

    卸载Manjaro

  11. 删除分区：此电脑->管理->磁盘管理，删除Manjaro的分区
  12. 下载DiskGenius，左侧目录，SYSTEM->EFI，右侧删除EFI文件夹下的manjaro文件夹
  13. 下载EasyBCD，修复引导：编辑引导菜单，保存设置（不勾选Use Metro bootloader）
    （EasyUEFI也可）
